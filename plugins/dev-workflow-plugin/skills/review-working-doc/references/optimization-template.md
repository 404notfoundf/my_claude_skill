# 优化建议清单模板

适用于当前项目中的文档整理、结构优化、样式优化、可维护性建议、非缺陷类改进项。

## 推荐骨架

```md
# 文档标题

一句话说明这份文档要优化什么。

说明：

- 这份文档基于什么来源整理
- 这里记录的是优化建议，不等同于安全漏洞或功能缺陷

## 总览

| ID | 优先级 | 优化点 | 建议类型 | 预期收益 | 备注 |
|---|---|---|---|---|---|
| 1 | 高 | 统一标题层级 | 可读性 | 更易维护 |  |
| 2 | 中 | 合并重复状态字段 | 结构简化 | 减少双写 |  |

## 详情

### 1. 统一标题层级

- 优先级：高

**现状问题**

- 说明当前结构哪里不顺

**优化建议**

- 说明建议怎么整理

**预期收益**

- 说明改完后会更好在哪里

**相关位置**

- `path/to/file.md`

## 推荐采纳顺序

1. 先处理影响阅读效率最大的项
2. 再处理结构一致性问题
3. 最后处理风格细节
```
